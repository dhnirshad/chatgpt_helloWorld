# chatgpt_helloWorld
Author: IRSHAD

Step-1: Install prerequisite
1. pip install openai
2. pip install python-dotenv

Step-2: Generate your openai key from https://platform.openai.com/account/api-keys.

Step-3: create .env file in same directory having your openapi key 
OPENAPI_KEY="put your openai key here"

Step-4: Execute and get your result-
```
python chatgpt_client.py

Hi Folks, Ask your question? What is the capital of India

Here is my response :
The capital of India is New Delhi.
```
